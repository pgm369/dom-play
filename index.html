Mean Girls Script

Home

I have created a JavaScript page for a single-page application that serves as a tablet interface for actors rehearsing a play based on Mean Girls. The application allows actors to touch their characters' names on the tablet to highlight their lines and un-highlight lines from other actors. It can handle any number of actors and lines and updates dynamically without reloading the page. The code utilizes the DOM and provides a user-friendly experience for efficient rehearsals.
